Vision & Grundidee

Das Kernproblem im Printmarkt heute

Klassische Bildbeschaffung für Print-Medien ist langsam, teuer und unflexibel. Ein professionelles Produktfoto kostet zwischen 150 und 600 €, ein Fotoshooting-Tag 2.000–6.000 €. Stockbilder sind generisch. Individualisierung kostet extra.

Gleichzeitig explodiert der Bedarf durch E-Commerce, Print-on-Demand, Personalisierung und Multichannel-Marketing — besonders im deutschen Markt mit über 500.000 aktiven Online-Shops.

Die Lösung: AI Media Factory für Deutschland

Eine vollautomatisierte KI-gesteuerte Bildfabrik, die für Print, Web, POD-Shops und Werbung maßgeschneiderte Bilder generiert — in Druckqualität (CMYK, 300 DPI, TIFF), mit lückenlosem ERP-Tracking.

Zwei Geschäftsmodelle parallel

B2B: Agentur-Service

Firmen, Verlage und Agenturen beauftragen Bilder per Brief oder API. Abrechnung per Flatrate oder Einzelbild.

Marge: 80–90%

B2C: Print-on-Demand Shops

Eigene Produkte (Poster, T-Shirts, Karten) auf Etsy, Amazon, Redbubble verkaufen. Passiveinnahmen durch Volumen.

Marge: 40–70%

Langfristige Vision

Keine klassische Agentur, keine gewöhnliche Softwarefirma. Sondern eine AI-gestützte visuelle Produktionsinfrastruktur — vergleichbar mit einer vollautomatisierten Druckerei, die on-demand Bilder und Druckprodukte generiert.

Langfristig: SaaS-Plattform mit White-Label für Druckereien und POD-Anbieter.

Primäre Zielbranchen Deutschland

1. Print-on-Demand / E-Commerce (B2C)

  • ~80.000 aktive Etsy-Shops in DE, Wachstum +35%/Jahr
  • Amazon Merch: 500 Mio. Produkte, täglich tausende neue Designs gefragt
  • Nischen mit wenig Konkurrenz: regionale Motive, Feiertage, Berufs-Nischen

2. Werbeagenturen & Druckereien

  • ~12.000 Werbeagenturen in Deutschland (Stat. Bundesamt 2024)
  • Budgets sinken, Bildbedarf steigt → perfekter KI-Use-Case
  • Kontaktwege: LinkedIn, BDW (Berufsverband), lokale Netzwerke

3. Grußkarten & Papeterie

  • Geburtstag, Weihnachten, Hochzeit — immer gefragt, immer wiederkehrend
  • Personalisierung als Trend: "Karte mit eigenem Haustier-Motiv" → LoRA-Training

4. Mode / T-Shirt-Business

  • Print-on-Demand über Printful, Printify, Spreadshirt
  • Design erstellen → dropshipping, kein Lager nötig
  • Nischen: Hobbys, Berufe, Regionen, Humor

Deutscher Markt — Besonderheiten & Rechtliches

  • DSGVO: Kundendaten auf EU-Servern (Hetzner DE), Datenschutz-Hinweis auf Website obligatorisch
  • MwSt.: 19% USt auf digitale Leistungen, Kleinunternehmerregelung bis € 22.000/Jahr möglich
  • Urheber-KI-Recht (neu 2025): AI-generierte Werke müssen bei kommerzieller Nutzung als KI-erstellt gekennzeichnet werden (AI Act)
  • Impressumspflicht: Auch für Etsy-Shops mit Wohnsitz DE erforderlich
  • Preissensibilität: DE-Markt preissensibel bei B2C, bei B2B aber qualitätsorientiert → zwei Preisstrategien nötig

Hinweis: POD-Dienste (Printful, Redbubble, Spreadshirt) arbeiten mit RGB — kein CMYK-Export nötig!

Python CMYK-Konverter (Produktionscode)

# image_generator/utils/color_converter.py
from PIL import Image, ImageCms
import os

ICC_PROFILES = {
    "FOGRA51": "/usr/share/color/icc/PSOcoated_v3_FOGRA51.icc",
    "FOGRA52": "/usr/share/color/icc/PSOuncoated_v3_FOGRA52.icc",
    "FOGRA39": "/usr/share/color/icc/ISOcoated_v2_eci.icc",
}

def convert_rgb_to_print_cmyk(input_tiff_path, output_tiff_path, profile_key="FOGRA39"):
    img = Image.open(input_tiff_path)
    if img.mode != "RGB":
        img = img.convert("RGB")
    srgb = ImageCms.createProfile("sRGB")
    target = ICC_PROFILES.get(profile_key)
    if not target or not os.path.exists(target):
        raise FileNotFoundError(f"ICC-Profil '{profile_key}' nicht gefunden")
    cmyk_profile = ImageCms.getOpenProfile(target)
    transform = ImageCms.buildTransform(
        srgb, cmyk_profile, "RGB", "CMYK",
        renderingIntent=ImageCms.INTENT_RELATIVE_COLORIMETRIC
    )
    cmyk_img = ImageCms.applyTransform(img, transform)
    cmyk_img.save(output_tiff_path, format="TIFF",
                  compression="tiff_lzw", icc_profile=cmyk_profile.tobytes())
    return output_tiff_path

  • Vor Valentinstag: Liebes-Poster-Set automatisch erstellen
  • Cron-Job im ERP plant saisonale Batch-Jobs 8 Wochen im Voraus

  • Kunden laden 20–30 Referenzbilder hoch
  • System trainiert LoRA automatisch (Kohya SS)
  • Alle zukünftigen Jobs nutzen das Kunden-LoRA → Markenkonformität

Etsy — Strategie im Detail

Shop-Aufbau

  • Shop-Name: Nischenbezogen (z.B. "NordlichterDesign")
  • Banner + Profilbild professionell mit AI generieren
  • Mindestens 20 Listings beim Start für Sichtbarkeit
  • Produkttitel: SEO-Keywords vorne (z.B. "Poster Berlin Skyline Wandbild A3 druckbar")
  • Tags: alle 13 erlaubten Tags nutzen (Long-tail Keywords)

Digitale Downloads (höchste Marge)

  • PDF oder PNG-Datei → sofortiger Download, kein Versand
  • Preis: € 3–15 pro Design
  • Mehrsprachige Listings (DE + EN für internationale Käufer)
  • Bundle-Listings: "10er Poster-Set Boho" → höherer Warenkorb
  • Kennzeichnung: "KI-generiertes Design" (Etsy-Pflicht)

Django → Etsy-Automatisierungspipeline

# Celery Task: auto_publish_to_etsy
# 1. Design generieren (FLUX) + Upscale auf 300dpi
# 2. Mockup erzeugen (Poster in Wohnzimmer, T-Shirt auf Model via Printful Mock API)
# 3. Etsy Listing via Etsy API v3 erstellen:
#    - Titel, Tags, Beschreibung aus Prompt-Template
#    - 5–10 Produktbilder (Design + Mockups)
#    - Preis + Kategorie
# 4. Asset-Eintrag in Django DB mit etsy_listing_id speichern

Amazon Merch on Demand

  • Was ist das? Amazon produziert und versendet T-Shirts, Hoodies, Tassen auf Anfrage — komplett passiv
  • Royalty: ~ 15–25% pro Verkauf (bei € 20 T-Shirt: ~€ 3–5 Royalty)
  • Zugang: Bewerbung nötig (Warteliste, Tier-System 10→25→100→500→1000+ Slots)
  • Strategie: Nischen-Designs mit wenig Konkurrenz, hohe Suchvolumina (z.B. "Dackel Mama" statt "Hund")
  • Django-Integration: Designs als PNG 4500×5400px exportieren, manuell oder über inoffizielle Tools hochladen

Redbubble & Society6 — Passiver Einkommensstrom

Redbubble

  • 1 Design → 60+ Produkttypen automatisch
  • Royalty: 20% Aufschlag auf Basispreis frei einstellbar
  • Keine Shop-Gebühren, keine Mindestanzahl
  • Starke internationale Community (USA, AUS, UK)
  • Schwäche: kein DE-Lager → 10–14 Tage Lieferzeit

Society6

  • Premium-Kunstposter, Leinwände, Wohnaccessoires
  • Höherpreisige Zielgruppe (Kunstaffinität)
  • Royalty: 10% (Poster), 5–10% (andere Produkte)
  • Gut für: Botanische Prints, Abstrakte Kunst, Architektur

Eigene Website / Shopify — Volle Kontrolle

  • Plattform: Shopify Basic (€ 27/Monat) oder WooCommerce (selbst gehostet)
  • Fulfillment: Printful-Plugin → automatischer Druck & Versand
  • Vorteil vs. Etsy: Keine Provision (nur Stripe ~2,9%), eigene Kundendaten, Branding
  • SEO-Strategie: Blog-Beiträge ("Beste Poster für minimalistisches Wohnzimmer"), Pinterest-Pins
  • E-Mail-Liste: Rabatt-Code als Lead-Magnet → Newsletter für Wiederkäufer
  • Django-Backend Integration: Webhook bei Bestellung → Generierung kundenspezifischer Variante → Printful-Upload

Akquise-Prioritäten (Deutschland)

  • SOFORT: Etsy digitale Downloads starten — kein Kapital nötig, sofortige Einnahmen möglich
  • SOFORT: Redbubble-Account anlegen — 50 Designs hochladen, passiv laufen lassen
  • MONAT 2: Kleine Werbeagenturen DE (5–20 MA) — direkter ROI-Nachweis möglich
  • MONAT 3: E-Commerce-Händler auf Amazon/Shopify — hohe Wiederholungsrate
  • MONAT 4: Shopify-Eigenshop + Printful — kein Etsy-Anteil mehr
  • MONAT 5: Adobe Stock / Shutterstock — passive Lizenzeinnahmen aufbauen
  • MONAT 6+: FLYERALARM / Saxoprint White-Label-Partnership — langer Saleszyklus

3-Jahres-Vision Deutschland

Jahr 1: System aufgebaut, 3–8 B2B-Kunden, Etsy + Redbubble aktiv, Einnahmen € 2.000–6.000/Monat

Jahr 2: SaaS-Portal Self-Service, 200+ Etsy/Amazon-Listings, Shopify-Eigenshop, € 8.000–20.000/Monat

Jahr 3: White-Label für 2–3 Druckereien, Lizenzeinnahmen Stock-Plattformen, ARR > € 150.000

Exit-Option: Verkauf des Systems (Infrastruktur + Kundenstamm + trainierte Modelle) an Druckerei-Kette oder Mediahaus. Bewertet nach 2–3× Jahresumsatz.
